# Interview Guide

Explain this project as an AI platform, not just a chatbot:

- It operationalizes RAG with retrieval, ranking, generation, evaluation, and governance.
- It uses MCP-style tools to make agent capabilities discoverable and testable.
- It keeps AI providers abstract so mock providers power CI and OpenAI-compatible providers can run in production.
- It includes observability through structured logs, trace IDs, audit logs, health checks, and metrics.
- It uses Docker and CI to show production engineering habits.
